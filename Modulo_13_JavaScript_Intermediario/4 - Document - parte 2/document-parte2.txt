O DOM é uma representação em memória do documento HTML ou XML e permite ao JavaScript interagir com os elementos do documento. Isso significa que o JavaScript pode acessar, criar, modificar e excluir elementos e conteúdo na página da web usando o DOM.
Usando a propriedade innerHTML, você pode obter ou definir o conteúdo HTML de um elemento. Por exemplo, você pode usar o innerHTML para inserir um novo conteúdo HTML em um elemento existente ou para modificar o conteúdo atual de um elemento.
