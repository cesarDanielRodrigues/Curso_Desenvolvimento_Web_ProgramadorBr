Para criar lista é necessario usar a tag<ul> ou a tag<ol>, ul seria para uma lista sem ordem enquanto ol seria para uma lista com ordem(1,2,3...). Para cada item da lista você tem que usar a tag<li> (Para fazer vários <li> de uma só vez, usar li*(numero de vezes));
É possível adicionar atributos dentro das tag<ul> ou tag<ol>, atributos como "reverse" que reverte a ordem da lista, o atributo " start="" " que determina um numero ou posição vai começar a lista(funciona apenas no <ol>), e o atributo "type=""" que define o tipo de numero ou posição que vai ser usado pela lista (funciona apenas no <ol>)
