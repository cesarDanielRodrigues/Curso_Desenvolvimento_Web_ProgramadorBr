Para buscar por link,telefone,whatsapp,email e a grande maioria dos links, é necessário usar a tag <a href="link">, em casos como telefone tem que colocar <a href="tel:+999999999999">, enquanto o whatsapp segue um padrão de link diferente que pode ser link para uma pagina do whatsapp ou até msm aplicação do whatsapp, o instagram a msm coisa, por isso é interresante analisar as aplicação possíveis(instagram inclusive se utiliza paramentros), e o email é utilizado <a href="mailto:exemplo@gmail.com"> para já direcionar para a aplicação de email;
O atributo target="_blank" serve para que o link envie para uma nova janela onde será feita toda aplicação.
