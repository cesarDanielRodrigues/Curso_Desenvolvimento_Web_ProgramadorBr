A estrutura básica é formada por <!DOCTYPE html> que é para definir qual a versão do HTML, <html> e seus filhos, <head> e <body> (os filhos são tags dentro de outras tags, as tags de dentro são chamadas de filhos e as que contem essa tags de pai), dentro da tag <head> vai tudo que não é visivel dentro da página como o título da página, seu icon, entre outros dados, já dentro da tag <body> vai tudo o que é visível dentro do site como oss textos, parágrafo, os títulos, imagens, entre outros.
