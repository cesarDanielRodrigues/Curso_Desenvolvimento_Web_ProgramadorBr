Nos ajustes de texto, as propriedades css são modificar diversas formas do texto
Essas propriedades são:
color - Altera a cor do texto
text-transform - Altera a apresentação do texto
text-decoration -  Altera a decoração do texto, e possível adicionar mais de uma parametro na mesma propriedada
text-align - Altera o alinhamento do texto
letter-spacing - Altera o espaçamento entre as letras
word-spacing - Altera o espaçamento entre as palavras
line-height - Altera o espaçamento de entre linhas
text-indent - Altera a indentação do paragrafo
text-shadow - Adiciona sombra ao texto, para isso ele possui 4 parametro(deslocamento para esquerda,deslocamento para baixo, intencidade da cor e cor)
